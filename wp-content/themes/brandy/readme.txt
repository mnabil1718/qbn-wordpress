== Description ==

Brandy is an elegant and versatile WordPress theme designed for WooCommerce to help you create a stunning online store with ease. Whether you're a web agency, freelancer, or entrepreneur, Brandy has everything you need to showcase your products and connect with your customers.

The Brandy theme and plugins offer a clean and elegant design that looks fantastic on all devices (desktop, tablet, and mobile). You can choose to use Brandy with Gutenberg block editor or advanced page builders like Elementor.

It is designed for easy customization. Any WordPress user can customize the ecommerce website with a variety of theme options and a user-friendly interface. No coding required!

= Features =

* WooCommerce Ready: Looking to sell products online? Brandy seamlessly integrates with WooCommerce for a super smooth ecommerce experience.
* SEO Optimized: Built with clean code and search engine optimization in mind to help your website rank higher in search results.
* Child Theme Compatible: Create child themes to further customize your website without affecting core theme files.
* Conversion Oriented: It comes with the most practical functions to help you sell better and get more customers.
* Frequent Updates: Enjoy free lifetime updates with new features/bug fixes for the core theme and plugins.

== Getting Started ==

= Installation =
1. From the WordPress admin dashboard, go to **Appearance** > **Themes** > **Add New**
2. Search for "Brandy"
3. Install and activate the theme.

= Theme Options = 
Navigate to the "Brandy" menu to access Brandy's theme options panel. Here you can customize headers, footers, colors, fonts, layouts, and more.

= Documentation = 
For detailed instructions and tutorials, refer to the comprehensive documentation sidebar in Brandy's dashboard.
